# Global-Post-Covid-Inflation-Trends-Project-

## Project Overview

This project analyzes the relationship between inflation rate, economic growth, and policy responses after COVID-19.

---

## Objectives
- Inflation trends across selected economies
- Economic drivers and policy responses
- Key insights from GDP, employment, and price indicators
- Strategic recommendations for inflation management

## Tools Used
1. Microsoft Excel - Data Cleaning
2. Power BI - Data Visualization, Data Modelling, DAX
3. Power Point - Report Writing

---
## Project Structure
```
├──global_inflation_post_covid_Data.csv         # Raw Data
├──global_inflation_post_covid.csv              # Cleaned Data
├──Global Post COVID Economic Trends.pbix       # PoweBI Dashboard  
└──Economic Reports.pptx                        # PowerPoint Reports
```
---

## Insights
- Economic recovery supported growth
- Inflation surged during the recovery phase 
- Food prices rose consistently
- Interest rates played a moderating role
- Exchange rates remained relatively stable
- Currency fluctuations had a limited impact on inflation.

## Recommendations
1. Maintain prudent monetary policy to keep inflation stable
2. Strengthen food supply chains to reduce food-price volatility.
3. Promote sustainable economic growth without creating excessive inflationary pressure.
4. Improve exchange-rate stability to limit imported inflation.
5. Continue monitoring key economic indicators for early detection of inflation risks.
